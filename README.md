## Overview

Cashlytics is a personal finance tracker that lets you manage accounts, track income and expenses, set budgets, log transactions, and view upcoming events on a calendar.

## How to Download
Cashlytics is a PWA (Progressive Web App), which means you can install it like a normal app on your phone or computer.
> **Note:** Once installed, Cashlytics will continue working offline.

### Install on Desktop
1. Open the [Live Demo](https://josiah-j-projects.github.io/Cashlytics/)
2. Look for the **Install** icon in the address bar
3. Click **Install**

### Install on Android

1. Open the [Live Demo](https://josiah-j-projects.github.io/Cashlytics/)
2. Tap the 3 dot **"⋮"** menu
3. Select **Add to Home Screen**
4. Select **Install**

The app will now appear on your home screen.

### Install on iPhone/iPad

1. Open the [Live Demo](https://josiah-j-projects.github.io/Cashlytics/)
2. Tap the 3 dot **"⋯"** menu
3. Tap the **Share** button
4. Scroll down and tap **Add to Home Screen**
5. Tap **Add**

The app will now appear on your home screen.

## Features

| Feature | Details | Implemented |
|--------|---------|-------------|
| Dashboard | Overview of balances, activity, and key financial insights in one view. | ✓   |
| Accounts | Manage account types. | ✓   |
| Income Streams | Set up recurring income with customizable options. | ✓  |
| Credit Accounts | Track credit balances, interest, grace periods, and minimum payments automatically. | ✓   |
| Recurring Expenses | Schedule recurring expenses linked to accounts and budget categories. | ✓ |
| Budgeting Page | Allocate budgets by category with visual breakdowns and spending comparisons. | ✓ |
| Transaction Log | View and search all transactions with filtering and per-account history. | ✓ |
| Calendar | See upcoming financial events and add transactions directly from a monthly view. |✓ |
| Save and Load | Export and import data with the option of encrypting the data. | ✓ |
| Catch Up Data | Automatically generate missed recurring transactions when loading older data. | ✓ |
|Financial Reorganisation| Transfer money between accounts, Recurring Account transfers, account trannsfers can be assigned to budget categories such as for investments or emergency funds|✓|
| Demo Mode | Load sample data for demonstration purposes. |✓|
| PWA Support | Add PWA support so that the app can be downloaded on both desktop and mobile|✓|
| Backend Database | Persistent storage for securely saving and retrieving user data anywhere. | |
|Auto Calc|Option to automatically calculate/suggest certain values such as ideal emergency fund amount and ideal amount to allocate towards debt||
|Custom Dashboard| Allow users to add/remove whatever metrics they would like on the dashboard ||
